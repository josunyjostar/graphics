shift + alt + 좌클릭 : 멀티 커서 범위 선택

end 키 : 해당라인 맨 뒤

home + shift : 커서 기준 앞으로 라인 드래그

home + end : 커서 기준 뒤로 라인 드래그

ctrl + shift + f9 : 모든 중단점 제거

alt + shift + . : vscode 에서 ctrl + d랑 같은 기능
